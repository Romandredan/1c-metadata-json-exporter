# История изменений

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.1.0/),
проект следует [семантическому версионированию](https://semver.org/lang/ru/).

## [Не выпущено]

### Планируется
- Блок СвойстваОбъекта (вид регистра, тип иерархии, периодичность)
- Блок Связи (регистраторы, владельцы, журналы)
- Свойства конфигурации в корне JSON
- Карта API модулей

## [1.0.0] — 2026-05-27

### Добавлено
- Универсальная выгрузка справочников, документов, регистров, перечислений, ПВХ, ПВР, планов счетов
- Три режима: один объект / список / подсистема
- Программный API
- Раскрытие ссылочных типов с защитой от рекурсии
- Опция компактного JSON
- Регистрация как дополнительная обработка БСП
